# Key Files

Important files in a Cpx project and their purposes.

## vcpkg.json

vcpkg manifest file that lists all dependencies. Managed via `cpx add port` commands.

## CMakeLists.txt

Main CMake build file with vcpkg integration. Auto-generated by `cpx create`.

## CMakePresets.json

CMake presets for IDE integration. Uses environment variables, safe to commit.

## cpx.ci

Cross-compilation configuration for Docker-based builds. Created with empty targets.

## .clang-format

Code formatting configuration. Auto-generated based on your style preference.

## .gitignore

Git ignore file with common build artifacts and IDE files.

# Quick Start

Get started with Cpx in minutes!

## Installation

```bash
curl -f https://raw.githubusercontent.com/ozacod/cpx/master/install.sh | sh
```

## Create Your First Project

```bash
# Create an executable project
cpx create my_app

# Create a library project
cpx create my_lib --lib

# Create from a template
cpx create my_project --template default
```

## Build and Run

```bash
cd my_app
cpx build
cpx run
```

That's it! You're ready to start building C++ projects with Cpx.
